How this works

Invoices always route buyer → manager → accountant, in that order. The accountant step is filled automatically since any accountant can handle it.

Reviewers act one at a time, in the order listed. The document moves to the next reviewer only after the current one approves.

If any reviewer rejects, the workflow stops and the document is marked rejected.

You can send reminders and follow the full timeline from the workflow page once it's started.
